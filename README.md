# 🌍 Wonderlust – Full Stack Travel Platform

Wonderlust is a full-stack web application inspired by Airbnb that allows users to explore, list, and book rental properties. It provides a seamless experience for both property owners and travelers.

---

## 🚀 Features

* 🏡 Browse rental listings
* 🔍 Search and filter properties
* 📝 Add, edit, and delete listings
* 👤 User authentication (Login / Signup)
* 💬 Reviews and ratings system
* 📍 Location-based listings
* 📷 Image upload for properties

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript
* EJS (Embedded JavaScript Templates)

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB (Mongoose)

**Other Tools:**

* Cloudinary (for image upload)
* Map integration

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/rushikesh7499/Wonderlust.git
```

2. Navigate to project folder:

```bash
cd Wonderlust
```

3. Install dependencies:

```bash
npm install
```

4. Create a `.env` file and add:

```
MONGO_URL=your_mongodb_connection
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
